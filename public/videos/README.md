# Introduction video

Drop the introduction film here so the home-page video section
picks it up automatically.

| File              | Used by                              |
| ----------------- | ------------------------------------ |
| `intro.mp4`       | `<IntroVideo />` on the home page    |
| `intro.webm`      | Optional WebM fallback               |
| `intro-poster.jpg`| Poster frame shown before play       |

Recommended: 1080p, ≤60s, no audio (autoplay friendly), under
~10 MB. The component renders a placeholder card and a "Coming
soon" caption while these files are missing.
